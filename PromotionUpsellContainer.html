Class: PromotionUpsellContainer

PromotionUpsellContainer

Represents a container that contains the promotion upsell message list.

new PromotionUpsellContainer(promotionUpsellMessages, isNonQualifiedMessagesRetrievalInProgress, isNonQualifiedMessagesSubscribedToQuantity)

Creates a PromotionUpsellContainer.
Parameters:
Name Type Description
promotionUpsellMessages Array List of promotion messages.
isNonQualifiedMessagesRetrievalInProgress boolean Indicates if an endpoint call is currently under progress to fetch the non qualified promotions. If set to true, it will not initiate similar calls.
isNonQualifiedMessagesSubscribedToQuantity boolean Indicates if the endpoint calls have been subscribed to changes in cart quantity. This variable is introduced to SKIP nonQualifiedPromotionMessage endpoint call from multiple widget instances across the site.
