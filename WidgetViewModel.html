Class: WidgetViewModel

WidgetViewModel

Represents a widget.

new WidgetViewModel(basePath)

The WidgetViewModel is a view model representing a widget object. Each widget displays or interacts with page data and fills a region. A WidgetViewModel may represent a widget definition, a default widget object that can be instantiated & loaded, or a widget instance, the instantiation of a widget with configuration and data.
